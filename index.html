Multi-Object Quality Control

This project aims to improve product quality control by identifying and categorizing defective chips and chocolates through a combination of deep learning, object detection, and robotic arm control. The camera captures an image of the product, which is analyzed by a YOLO algorithm .The Delta robotic arm is then directed to pick and place the defective product into a designated basket, enhancing efficiency and reducing human error. This system improves production quality, minimizes wastage, and ensures the delivery of high-quality productsÂ to consumers.

Work Plan

Data Preperation

RoboFlow

Feb-March

Dataset Preparation: A comprehensive dataset comprising images of both chips and chocolate was curated. This dataset serves as the foundation for training and evaluating the object detection and classification model.The dataset images were annotated with bounding box coordinates corresponding to the locations of chips and chocolate objects within each image in RoboFlow Platform.

Model Selection & Implementation

Yolo V5

Apr-Aug

Model Training: The YOLOv5 model was trained using the annotated dataset. The model learned to detect and classify chips and chocolate based on the provided bounding box coordinates. The YOLOv5 model was configured to perform multi-class object detection and classification. Specifically, the model was trained to distinguish between chips and chocolate, making it capable of recognizing and categorizing both types of items in images.

Efficient Sorting System

Conveyor Belt

Sep-Nov

YOLOv5 transforms chip and chocolate sorting on a conveyor belt, providing precise quality classification with real-time object detection and coordinate outputs. This advanced technology enhances efficiency and quality control by swiftly identifying and pinpointing the exact location of items, contributing to a seamless production process.

Segregration using Delta Robotic Arm

In Progress

Aims to identify defective products based on pre-defined criteria post-production and convey the product's coordinates to the Delta robotic arm and It also involves segregating the defective product by picking it up from the received coordinates, thus enhancing the overall quality.
